# Amazon_Vine_Analysis

## 1. Overview 
To analyze the influence of reviews of users enrolled in the amazon vine program.
Dataset: https://s3.amazonaws.com/amazon-reviews-pds/tsv/amazon_reviews_us_PC_v1_00.tsv.gz

## 2. Results:
